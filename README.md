# jonotype

boilerplate react app without using create-react-app

removes all the fluff!

## structure

- .gitignore
- package.json
- public
  - index.html
- src
  - index.js
  - App.js
  - Component.js
  - NewComponent.js

## usage

```
$ git clone https://github.com/joncoded/jonotype.git
$ npm install
$ npm start
```
